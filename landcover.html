Land Cover Classification /h1>

The Assignment

The aim of this assignment is to gain experience in generating land cover classification maps using satellite imagery and remote sensing techniques. Given that there are countless different ways to classify land cover, this process is a key part of environmental monitoring. The different techniques learned include supervised pixel-based classification, unsupervised object-based classification and unsupervised pixel-based classification. Supervised pixel-based classification allows the user to define specific land cover classes by providing training samples, from which the algorithm then classifies all the pixels in the image. Unsupervised object-based classification, on the other hand, groups pixels into objects (or segments) based on their spectral and spatial properties before classifying these objects into land cover classes without using user-defined training samples. Lastly, unsupervised pixel-based classification automatically groups pixels into clusters based on their spectral properties without any prior knowledge or training samples from the user. The assignment will cover the three techniques as well as the normalized difference vegetation index (NDVI).
